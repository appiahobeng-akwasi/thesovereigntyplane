The audit trail.

Every score change in the master tracker, with date, scorer, trigger, and rationale. This is the page reviewers can use to challenge whether a score moved appropriately. It is also the defence if anyone asks how the data evolved.

How version control works. The master_tracker.xlsx file is archived with a timestamp before each material refresh. The current file's Version_Log sheet captures every indicator-level change. To roll back, restore the archive copy. To add a change, edit the score and append a row.
