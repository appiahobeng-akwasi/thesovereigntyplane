# 3D Immersive Intro Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a full-viewport WebGL fly-through intro that plays on page load, flying through floating country nodes and landing on a visual sovereignty plane before revealing the existing page.

**Architecture:** IntroGate.tsx is a React component mounted with `client:load` on index.astro. It detects viewport width and renders either IntroScene.tsx (R3F WebGL, desktop) or IntroMobile.tsx (2D canvas, mobile). The intro sits as a fixed overlay above the existing page, which renders normally behind it. On completion, the overlay fades out and unmounts.

**Tech Stack:** React Three Fiber, drei, postprocessing, Three.js, 2D Canvas API

```typescript
import type { Country } from '../data/types';
import { quadrantColor } from './plane-geometry';

// Animation phase durations (seconds)
export const PHASE_1_END = 1.5;
export const PHASE_2_END = 3.0;
export const PHASE_3_END = 4.5;
export const SKIP_FADE_MS = 300;

// Camera positions
export const CAMERA_START = [0, 5, 50] as const;
export const CAMERA_END = [0, 40, 0] as const;

// Scene
export const SCENE_BG = '#0a0a0f';
export const PARTICLE_COUNT = 500;
export const PARTICLE_SPREAD = 60;
export const NODE_BASE_SIZE = 0.3;

// Mobile breakpoint
export const MOBILE_BREAKPOINT = 768;

// Map a country to a random 3D starting position
export function randomStartPosition(index: number, total: number): [number, number, number] {
  const phi = (index / total) * Math.PI * 2;
  const r = 8 + Math.random() * 12;
  const y = (Math.random() - 0.5) * 10;
  return [Math.cos(phi) * r, y, Math.sin(phi) * r - 20];
}

// Map a country to its target 2D plane position in 3D space
// The plane surface is 20x20 units, centered at origin, at y=0
export function planePosition(country: Country): [number, number, number] {
  const x = (country.formal_score / 100) * 20 - 10;
  const z = (country.substantive_score / 100) * -20 + 10;
  return [x, 0.1, z];
}

// Get node color from country quadrant
export function nodeColor(country: Country): string {
  return quadrantColor(country.quadrant);
}

// Get node size based on gap score
export function nodeSize(country: Country): number {
  const gap = Math.abs(country.gap);
  return NODE_BASE_SIZE + (gap / 100) * 0.4;
}
```

### Task 3: Create IntroOverlay.tsx

**Files:**
- Create: `src/components/IntroOverlay.tsx`

- [ ] **Step 1: Create the overlay with skip button**

```tsx
import { useCallback } from 'react';

interface IntroOverlayProps {
  onSkip: () => void;
}

export default function IntroOverlay({ onSkip }: IntroOverlayProps) {
  const handleSkip = useCallback((e: React.MouseEvent | React.KeyboardEvent) => {
    e.stopPropagation();
    onSkip();
  }, [onSkip]);

  return (
    <div
      style={{
        position: 'absolute',
        inset: 0,
        zIndex: 10,
        pointerEvents: 'none',
      }}
    >
      <button
        onClick={handleSkip}
        style={{
          position: 'absolute',
          bottom: 40,
          right: 40,
          pointerEvents: 'auto',
          background: 'rgba(255,255,255,0.1)',
          border: '1px solid rgba(255,255,255,0.2)',
          color: 'rgba(255,255,255,0.6)',
          fontFamily: "'JetBrains Mono Variable', monospace",
          fontSize: 10,
          letterSpacing: '0.1em',
          textTransform: 'uppercase' as const,
          padding: '8px 16px',
          cursor: 'pointer',
          borderRadius: 2,
          backdropFilter: 'blur(4px)',
          transition: 'color 0.2s, border-color 0.2s',
        }}
        onMouseEnter={(e) => {
          e.currentTarget.style.color = 'rgba(255,255,255,0.9)';
          e.currentTarget.style.borderColor = 'rgba(255,255,255,0.4)';
        }}
        onMouseLeave={(e) => {
          e.currentTarget.style.color = 'rgba(255,255,255,0.6)';
          e.currentTarget.style.borderColor = 'rgba(255,255,255,0.2)';
        }}
      >
        Skip
      </button>
    </div>
  );
}
```
